# Discord-Generator
Discord Accounts Generator Written in Golang

## Features:
+ Email verifier
+ Avatar changer
+ Proxies support
+ Custom UserNames and Avatars

## Comming Soon
+ Phone Verification
+ Threads
+ Better error handling
